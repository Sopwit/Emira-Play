# Hackathon Submission Checklist

- English `README.md` exists at repository root.
- Soroban smart contract scaffold exists under `contracts/emira_rewards/`.
- Freighter wallet flow is implemented in the frontend.
- Stellar / Soroban environment variables are documented.
- Project screenshots are stored in `docs/screenshots/`.
- Verified test transaction hashes are stored in `docs/submission/transaction-hash.md`.
- Public production URL is documented as `https://emira-neaf.vercel.app`.
- Telegram Mini App launch details are documented for BotFather and QA.
- Build artifacts such as `dist/`, `coverage/`, and `node_modules/` are excluded from the public repository.
- Final GitHub repository link should be submitted on the Rise In platform after completion.
